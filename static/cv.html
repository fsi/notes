Опыт

OnlinePBX Ноябрь 2013 — Май 2017

Занимался разработкой, архитектурой, изучением новых технологий (с целью понять, имеет ли смысл их использовать), менторством и обучением новых программистов.

Основные проекты:

  • Поддержка и разработка расширения для браузеров Chrome и Firefox. Для Firefox расширение уже имелось, и надо было только его поддерживать. Для Chrome расширение писалось с нуля с использованием angular.js
  • Разработка приложения для холодного обзвона. Бэкэнд писался на PHP, фронтэнд — на jQuery и Bootstrap. Приложение состояло из админской и операторской частей.
    Впоследствии предпринималась попытка переписать фронт на angular.js, но ввиду малой популярности приложения его закрыли совсем.
  • Разработка виджета заказа обратного звонка для установки на сайты клиентов. Бэкэнд писался на PHP, фронтэнд - на чистом JavaScript для ускорения загрузки. Сейчас, пожалуй, использовал бы для такого Preact.
  • Участие в разработке виджетов для amoCRM. Не совсем моя заслуга, я лишь делал прототип. В том виджете, который используется сейчас, моего кода нет.
    Новую версию виджета (как раз на Preact) делала команда разработчиков с моим участием. Для сборки последней использовался rollup ввиду требований к сторонним виджетам в amoCRM.
  • Разработка микросервисной платформы для новой версии onlinePBX. Платформа на основе протокола AMQP 0-9-1, конкретно - RabbitMQ. Протокол общения и node.js модуль для его реализации. Приложения-микросервисы на node.js.
  • Разработка прототипа фронт-энда для нового продукта onlinePBX. Для разработки использовалась связка React+Redux. Для асинхронных действий применялись библиотеки redux-thunk, redux-promise, redux-saga. Протокол общения с бэкэндом — websocket (конкретно socket.io).
